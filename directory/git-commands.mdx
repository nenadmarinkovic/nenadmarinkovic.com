Git is a version control system that allows you to keep track of changes in your codebase, collaborate with other developers, and revert to previous versions of your code. Here are the most used Git commands that you need to know as a developer:

### git init

Initializes a new Git repository in your current working directory.

```shell
$ git init
```

### git clone

Clones a remote Git repository to your local machine.

```shell
$ git clone https://github.com/user/repo.git
```

### git add

Adds changes to the staging area in preparation for a commit.

```shell
$ git add file1.txt
```

### git commit

Commits changes to the repository.

```shell
$ git commit -m "Added new feature"
```

### git push

Pushes committed changes to a remote repository.

```shell
$ git push origin master
```

### git pull

Pulls changes from a remote repository to your local repository.

```shell
$ git pull origin master
```

### git branch

Lists all local branches in your repository.

```shell
$ git branch
```

### git checkout

Switches to a different branch in your repository.

```shell
$ git checkout branch-name
```

### git merge

Merges changes from one branch into another.

```shell
$ git merge branch-name
```
