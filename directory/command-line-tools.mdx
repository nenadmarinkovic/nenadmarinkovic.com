## Oh My Zsh

Oh My Zsh is an open source, community-driven framework for managing your Zsh configuration.
[Go to the website](https://ohmyz.sh/)

## fzf

A command-line fuzzy finder that allows you to quickly search for files and folders in your file system. You can integrate it with other command-line tools to improve your workflow.
[Go to the website](https://github.com/junegunn/fzf)

## tldr

A simplified and community-driven version of man pages that provides practical examples of how to use a command. It is great for quickly looking up commands without having to read through lengthy documentation.
[Go to the website](https://tldr.sh/)


## htop

An interactive process viewer for Linux that provides real-time monitoring of system resources. It is similar to the 'top' command but offers a more user-friendly interface.
[Go to the website](https://htop.dev/)


## bat

A cat clone with syntax highlighting and Git integration. It provides a more visually pleasing way to view files compared to the traditional 'cat' command.
[Go to the website](https://github.com/sharkdp/bat)


## ncdu

A command-line disk usage analyzer that allows you to quickly find out which files and directories are taking up the most space on your file system.
[Go to the website](https://dev.yorhel.nl/ncdu)


## exa

A modern replacement for the 'ls' command that provides more features and a more visually appealing interface. It includes options to show hidden files, sort files by various criteria, and much more.
[Go to the website](https://the.exa.website/)


## fd

A command-line tool that is used to find files and directories based on patterns. It is faster than other tools like 'find' and offers a more user-friendly interface.
[Go to the website](https://github.com/sharkdp/fd)


## fzy

A fuzzy text selector that allows you to quickly navigate through lists of text. It can be used to select files, processes, and much more.
[Go to the website](https://github.com/jhawthorn/fzy)

## jq

A command-line JSON processor that allows you to manipulate and transform JSON data. It supports a wide range of operators and functions, making it a powerful tool for working with JSON data
[Go to the website](https://stedolan.github.io/jq/)

## hub

A command-line tool that provides extra functionality to Git when working with GitHub. It allows you to create pull requests, fork repositories, and much more, all from the command-line.
[Go to the website](https://hub.github.com/)

## yq

A command-line YAML processor that allows you to manipulate and transform YAML data. It supports a wide range of operators and functions, making it a powerful tool for working with YAML data.
[Go to the website](https://github.com/mikefarah/yq)
