# 宖捉窶昨汳ｻ This is a  personal portfolio website created with [Next.js](https://nextjs.org/), [React](https://reactjs.org/), [styled-components](https://styled-components.com/).

#### Structure

- Banner with header
- Work
- Portfolio
- Technologies
- Footer with contact form

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:2500](http://localhost:2500) with your browser to see the result.
